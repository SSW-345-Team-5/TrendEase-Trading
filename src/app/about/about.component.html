This research project builds upon several concepts from data science & engineering, finance, and machine leanring. As a whole, the aim of this project is to create a valuable trading signal using a classifier, trained/tuned using an evolutionary algorithm.

Genetic Training/Tuning

In order for an algorithm to be considered evolutionary or genetic, it must uphold the following three properties of the natural selection process.

Heredity: The process by which children receive the properties of their parents. This allows for genetic material to be passed between generations.

Variation: There must be a variety of traits present in the population with which to introduce variation. High variation allows for the algorithm to explore the search space more fully to avoid settling at local extrema of the optimization problem.

Selection: The mechanism by which some members of the population have an opportunity to be parents and pass down their genetic information and some do not. This is the concept of “survival of the fittest” whereby some parents are more likely to pass on genetic information according to some fitness score.

Using these properties it is possible to then create a population of models with varying genetic material to explore the search space of various parameters and features of each model.
